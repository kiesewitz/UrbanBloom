# Design Tokens

Dieser Ordner ist für gemeinsame Design-Tokens vorgesehen, die von der Mobile App und Admin Web App verwendet werden.

## Geplante Inhalte

- **Farben**: Primär-, Sekundär- und Akzentfarben
- **Typografie**: Schriftarten, Schriftgrößen, Zeilenhöhen
- **Spacing**: Abstände und Padding-Werte
- **Breakpoints**: Responsive Design Breakpoints (für Web)
- **Shadows**: Schatten-Definitionen
- **Border Radius**: Abrundungen

## Format

Design Tokens können in verschiedenen Formaten gespeichert werden:
- JSON
- YAML
- CSS/SCSS Variablen
- Dart Konstanten

## Verwendung

Die Design Tokens werden von beiden Flutter-Apps (mobile & admin-web) importiert, um eine konsistente UI zu gewährleisten.
