# User Story US-CAT-02: Bestandsverwaltung und Inventarisierung

**ID:** US-CAT-02
**Titel:** Medien erfassen, aktualisieren und aussortieren
**Priorität:** Must-Have
**Epic:** Bibliotheksmanagement & Administration

**Als** Bibliothekar/in (Admin)
**möchte ich** neue Medien via ISBN-Scan erfassen, Stammdaten pflegen und alte Medien aussortieren,
**damit** der Bestand immer aktuell und vollständig ist.

**Akzeptanzkriterien:**
- [ ] Neues Buch erfassen via ISBN-Scan oder manueller Eingabe.
- [ ] System importiert Metadaten (Titel, Autor, Verlag, Klappentext) automatisch (falls verfügbar).
- [ ] Manuelle Ergänzung von Kategorie, Sprache, Standort, Signatur, Exemplaranzahl.
- [ ] Barcode-Druck für physisches Etikett.
- [ ] Bestandssuche und Bearbeitung einzelner Exemplare.
- [ ] Medien können als "aussortiert" markiert werden mit Datum und Grund.
- [ ] System schlägt vor: Medien, die seit 2+ Jahren nicht ausgeliehen wurden.
