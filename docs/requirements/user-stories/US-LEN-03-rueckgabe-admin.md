# User Story US-LEN-03: Rückgabe durch Administrator

**ID:** US-LEN-03
**Titel:** Medienrückgabe via Barcode-Scan
**Priorität:** Must-Have
**Epic:** Bibliotheksmanagement & Administration

**Als** Bibliothekar/in (Admin)
**möchte ich** Medien beim Rückgabevorgang schnell scannen und den Status aktualisieren,
**damit** das Buch sofort wieder verfügbar für andere Nutzer ist.

**Akzeptanzkriterien:**
- [ ] Scan des Medien-Barcodes zeigt Ausleih-Details (Nutzer, Rückgabedatum, Status).
- [ ] System prüft, ob Rückgabefrist überschritten ist (rot: überfällig).
- [ ] Rückgabe wird mit einem Klick verbucht und Status auf "Verfügbar" gesetzt.
- [ ] Überfällige Medien werden automatisch für Mahnprozess markiert.
- [ ] Erfolgsmeldung und sofortige Verfügbarkeitsaktualisierung.
