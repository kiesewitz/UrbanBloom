# User Story US-REM-01: Mahnwesen

**ID:** US-REM-01
**Titel:** Automatisierte Erinnerungen und Mahnungen für überfällige Medien
**Priorität:** Should-Have
**Epic:** Bibliotheksmanagement & Administration

**Als** Bibliothekar/in (Admin)
**möchte ich** automatische Mahnungen in mehreren Stufen versenden,
**damit** überfällige Medien zuverlässig eingetrieben werden.

**Akzeptanzkriterien:**
- [ ] System identifiziert täglich alle überfälligen Medien.
- [ ] Stufe 1 (nach 3 Tagen Überschreitung): Automatische E-Mail-Mahnung.
- [ ] Stufe 2 (nach 1 Woche): Zweite Mahnung mit erhöhtem Ton.
- [ ] Stufe 3 (nach 2 Wochen): Physischer Brief oder Verweis an Schulleitung (konfigurierbar).
- [ ] Mahnung enthält: Medien-Details, Überschreitungsdauer, Mahngebührenberechnung (falls konfiguriert).
- [ ] Bibliothekar kann Mahnung manuell auslösen oder zurückziehen.
- [ ] Nachlässe oder Verzeihe sind dokumentierbar.
