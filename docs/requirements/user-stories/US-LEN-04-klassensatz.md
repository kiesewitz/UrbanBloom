# User Story US-LEN-04: Klassensatz-Verwaltung

**ID:** US-LEN-04
**Titel:** Verwaltung und Buchung von Klassensätzen
**Priorität:** Should-Have
**Epic:** Kommunikation & Erweiterte Services

**Als** Lehrkraft
**möchte ich** Klassensätze buchen und verwalten,
**damit** meine Klasse zeitgerecht Medien für Unterrichtsprojekte erhält.

**Akzeptanzkriterien:**
- [ ] System erkennt Medien mit mehreren gleichen Exemplaren als "Klassensatz".
- [ ] Lehrkraft kann Klassensatz für einen Zeitraum reservieren (z.B. 4 Wochen).
- [ ] Reservierung zeigt die gewünschte Klassengruppe und Lernziele.
- [ ] Bibliothekar erhält Übersicht aller geplanten Klassensatz-Buchungen (Kalender).
- [ ] Automatische Benachrichtigung bei Änderungen (Verlegung, Stornierung).
- [ ] Alle Exemplare eines Klassensatzes werden zusammen aus- und eingecheckt.
