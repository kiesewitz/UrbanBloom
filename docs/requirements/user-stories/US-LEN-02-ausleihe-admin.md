# User Story US-LEN-02: Ausleihe durch Administrator

**ID:** US-LEN-02
**Titel:** Medienausleihe via QR Code-Scan
**Priorität:** Must-Have
**Epic:** Bibliotheksmanagement & Administration

**Als** Bibliothekar/in (Admin)
**möchte ich** Medien und Nutzer QR-Code auf dem Handy des Ausleihers scannen,
**damit** ich den Ausleihvorgang schnell und fehlerfrei verbuchen kann.

**Akzeptanzkriterien:**
- [ ] Scan des Nutzer-QR Code (oder manuelle Suche nach Nutzer) öffnet das Ausleihkonto.
- [ ] Scan des Medien-Barcodes fügt das Buch zur aktuellen Ausleihe hinzu.
- [ ] System prüft sofort auf Ausleihbeschränkungen (z.B. max. Anzahl erreicht, offene Mahngebühren).
- [ ] System zeigt das berechnete Rückgabedatum an (konfigurierbar je nach Nutzergruppe).
- [ ] Abschluss der Ausleihe mit einem Klick ("Verbuchen").
- [ ] Erfolgsmeldung und Aktualisierung des Bestandsstatus.
