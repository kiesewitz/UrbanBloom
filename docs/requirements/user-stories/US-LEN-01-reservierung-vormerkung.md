# User Story US-LEN-01: Reservierung & Vormerkung

**ID:** US-LEN-01
**Titel:** Medien reservieren und vormerken
**Priorität:** Must-Have
**Epic:** Digitale Ausleihe & Nutzerportal

**Als** Schüler oder Lehrkraft
**möchte ich** verfügbare Medien reservieren ("Click & Collect") oder verliehene Medien vormerken,
**damit** ich sie sicher erhalte, sobald ich in der Bibliothek bin oder sie zurückgegeben werden.

**Akzeptanzkriterien:**
- [ ] Bei verfügbaren Medien gibt es einen Button "Reservieren".
- [ ] Eine Reservierung ist für eine konfigurierbare Zeit (z.B. 2 Tage) gültig.
- [ ] Bei verliehenen Medien gibt es einen Button "Vormerken".
- [ ] Der Nutzer sieht seine aktuellen Reservierungen und Vormerkungen im Profil.
- [ ] Wenn ein vorgemerktes Buch zurückgegeben wird, wird es automatisch für den Vormerker reserviert.
- [ ] Ein Nutzer kann eine Reservierung/Vormerkung selbstständig stornieren.
